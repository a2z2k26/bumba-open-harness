# Changelog

All notable public changes to Bumba Open Harness are documented here.

The project uses [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and
will use semantic versioning once a stable public release is tagged.

## [Unreleased]

### Added

- Public adoption documentation for architecture, configuration, development,
  security posture, and job-search setup.
- OpenRouter-ready setup guidance for the backend registry.

### Changed

- Reset the public tree to a no-history publication baseline.
- Replaced private operator identity material with generic operator templates.
- Stubbed job-search configuration so adopters must supply their own candidate
  profile, criteria, links, and approval database.

### Removed

- Private planning archives, handoffs, local paths, personal job-search
  context, and runtime-only state from the public copy.
