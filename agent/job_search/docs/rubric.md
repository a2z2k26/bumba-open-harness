# Job Search Starter Rubric

This public rubric is a template for scoring incoming listings. Replace the
weights, threshold, and prompt fragments with the adopting operator's actual
criteria before running the job-search workflow live.

The machine-readable version lives at `agent/job_search/config/rubric.yaml`.
It produces a composite 1.0-5.0 score and letter grade. The default threshold
is intentionally conservative and should be tuned against the operator's own
target roles, compensation floor, location constraints, and domain interests.
