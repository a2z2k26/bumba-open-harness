# Candidate Context Template

Replace this file with the candidate profile that should inform cover-letter
generation, outreach research, and application form filling.

Recommended sections:

- Target roles and seniority
- Work authorization and preferred locations
- Relevant experience highlights
- Preferred domains, company stages, and technologies
- Resume, portfolio, LinkedIn, and GitHub URLs
- Topics or claims the assistant must not invent

Keep private contact data out of git. Store credentials in `.secrets` and put
candidate-specific artifacts in a local untracked `candidate.json`.
