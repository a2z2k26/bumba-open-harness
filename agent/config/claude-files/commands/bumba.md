---
description: Start a voice conversation with Bumba
---

Start a voice conversation using the `mcp__bumba__converse` tool. If the tool is not available, inform the user that the Bumba MCP server needs to be connected first.

IMPORTANT: The tool name is `mcp__bumba__converse`, NOT `mcp__chatta__converse`. The server was renamed from "chatta" to "bumba".
