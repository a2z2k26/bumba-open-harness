# Job Search Chief Artifacts

Write artifacts for job-search work that must be reviewable later.

Write artifacts for:

- application traces;
- sourced role lists;
- dedupe decisions;
- email verification findings;
- outreach drafts and staging notes;
- browser action traces when browser work was material.

Artifact rules:

- Write under the active Zone 4 run workspace.
- Redact credentials, tokens, cookies, and unnecessary personal data.
- Mark facts as verified, inferred, or blocked.
- Link the manifest and memory pointer in the final run summary.
- Never write application traces into the Bumba Mac repository by default.

If a run performs no material job-search action, do not create ceremonial
artifacts.
